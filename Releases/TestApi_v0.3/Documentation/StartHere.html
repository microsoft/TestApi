TestApi Documentation

The TestApi library is accompanied by the following documentation:

  • Concepts - this folder contains a number of conceptual documents.
  • API Documentation - provides MSDN-style documentation for all TestApi interfaces.
    • Available as a CHM file too.
For the most up-to-date documentation, please refer to the TestApi web site on http://codeplex.com/TestApi. Additional information on testing methodology is available in this document.
