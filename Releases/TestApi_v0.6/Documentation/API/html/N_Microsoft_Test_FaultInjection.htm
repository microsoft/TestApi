.
Microsoft.Test.FaultInjection Namespace
Send Feedback
Contains facilities for runtime fault injection in managed code. The API allows a test to simulate various faults in order to better exercise code-paths in the tested system.

Classes

  Class Description
BuiltInConditions
Contains all built-in fault injection conditions.
BuiltInFaults
Contains all built-in faults.
CallStack
Extracts frame information from a StackTrace object.
ComRegistrar
Provides facilities for registration and query of the fault injection COM engine.
FaultDispatcher
Calls the appropriate fault for a given faulted method.
FaultInjectionException
An exception that is thrown when and error in the FaultInjection API occurs.
FaultRule
Defines which method to fault, under what conditions the fault will occur, and how the method will fail. See also the FaultSession class.
FaultScope
FaultSession
Maintains information needed for injecting faults into a test application. For general information on fault injection see this page.
RuntimeContext
Stores information about a faulted method.

Interfaces

  Interface Description
ICondition
Defines the contract for specifying when a fault will be triggered on a method.
IFault
Defines the contract for a fault.
IRuntimeContext
Defines the contract for information provided by the faulted method.
